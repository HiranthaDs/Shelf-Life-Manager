# Expire FiFo 🛒📅  
A smart **inventory management system** with FIFO-based expiry tracking  

![GitHub License](https://img.shields.io/badge/license-MIT-blue.svg)  
![Python](https://img.shields.io/badge/python-3.x-blue)  
![Status](https://img.shields.io/badge/status-active-success)  

## 📌 Overview  
Expire FiFo helps businesses efficiently track product expiry dates using the **First In, First Out (FIFO)** method.  
🔹 **Reduce waste** by ensuring older products are used first  
🔹 **Easily manage inventory** with an intuitive UI  
🔹 **Automatic expiry alerts** for products near expiration  

---

## 🎯 Features  
✅ **Add, Update, and Remove Items**  
✅ **Expiry Date Tracking** (red highlight for expiring soon)  
✅ **Filter: Expiring Soon** (quickly view urgent items)  
✅ **Sort by Expiry Date**  
✅ **Password-Protected Deletion**  

---

## 🛠️ Tech Stack  
- **Python** – Backend logic  
- **PyQt / PySide** – GUI framework  
- **SQLite / MySQL** – Database  
- **PyInstaller** – Convert to executable
